Описание скрипта

Основные файлы
1. form.php - скрипт
2. requestTest.php - рабочий пример (поместить на сайт, передать данные в скрипт)
3. log.txt - логирование всех операций
4. client_id.txt - фаил с access и refresh токенами. Токены обновляются раз в 2 дня, перевыпуск автоматический
5. index.html - форма для ручного создания объявлений/тестов

Особенности

- Скрипт генерирует тело запроса на основе ключей из запроса. 
- Выполняется получение/запись в фаил и обновление токенов на основе схемы с refresh_token.

Важно!
- Информация по телу запроса которое нужно передать для целевого действия "САЙТ" предсавлена в файле requestTest.php
- Изображения должны иметь строго заявленые размеры!
- Файлы которые скрипт создает трогать не нужно.
